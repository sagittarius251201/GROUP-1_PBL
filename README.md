## Setup

1. Place `app.py`, `requirements.txt`, and `.streamlit/config.toml` at the root of your GitHub repo.
2. Commit & push to GitHub.
3. On Streamlit Cloud, set **App file path** to `app.py`.
4. Click **Redeploy**.
